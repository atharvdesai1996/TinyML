# TinyML — Machine Learning Assignments

A collection of hands-on assignments exploring core machine learning and deep learning concepts, progressing from basic neural networks to image classification with CNNs.

## Assignments

| Folder | Topic | Key Concepts |
|--------|-------|--------------|
| `DNN/` | Deep Neural Networks | Fashion MNIST, dense layers, ReLU, softmax |
| `LinearRegression/` | Linear Regression | TensorFlow basics, gradient descent, simple models |
| `Convolutions/` | Convolutions & Pooling | Feature extraction, filters, pooling layers |
| `BinaryClassification/` | Binary Image Classification | Sigmoid output, binary crossentropy, RMSprop, overfitting & augmentation |
| `CategoricalClassification/` | Multi-class Image Classification | Softmax output, categorical crossentropy, Adam, data augmentation |

## Learning Progression
1. **DNN** — Learn the basics of neural networks with dense layers on structured image data.
2. **LinearRegression** — Understand how TensorFlow optimizes simple models.
3. **Convolutions** — See how convolutional and pooling layers extract features from images.
4. **BinaryClassification** — Build a full binary image classifier (Horses vs Humans), understand overfitting and augmentation.
5. **CategoricalClassification** — Extend to multi-class problems (Bean Disease), learn categorical crossentropy and softmax.

## Tools & Frameworks
- Python, TensorFlow / Keras
- ImageDataGenerator for data loading and augmentation
- Matplotlib for visualization

## How to Use
Each folder contains:
- A Python script with the full training pipeline
- A README explaining the concepts and architecture choices
